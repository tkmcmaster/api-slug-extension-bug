## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

[API routes](https://nextjs.org/docs/pages/building-your-application/routing/api-routes) can be accessed on [http://localhost:3000/api/json/abcd1234.json](http://localhost:3000/api/json/abcd1234.json). 

```
curl http://localhost:3000/api/json/abcd1234.json

```
